## üîç Overview
This project is an **AI-powered search suggestion engine** that uses **Skip-Gram Word2Vec** embeddings trained on Wikipedia datasets. It generates real-time suggestions for input search queries by predicting contextually similar words.  

The goal is to simplify the **search experience** by providing intelligent suggestions.

## ‚ö° Features
- Generate suggestions for a given word/query.
- Trained on **full English Wikipedia** dataset for wide vocabulary coverage.
- Uses **Skip-Gram Word2Vec** for high-quality embeddings.
- Interactive interface with **search input and suggestion buttons**.
- Real-time predictions based on your query.
- Fully **Python-based**.
